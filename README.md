## Introduction
voyer.server is the backend for the voyer service. Voyeur is an analytics service.

## Dependencies

 - [nodejs](https://github.com/joyent/node)
 - [couchdb](https://github.com/apache/couchdb)
 - [phantomjs](https://github.com/ariya/phantomjs)
